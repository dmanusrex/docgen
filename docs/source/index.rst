|dg| is a utility for Club Official Chairs and Club Official Administrators to generate and optionally e-mail customized reports for each official in their club. 

Features
========

The |dg| system uses data exported from |SNC|_ |RTR|_ to generate customized development reports for each offiical in a club. |dg| also has the option to email the reports to each individual official. 

General Uses
============

- Start of season report to all officials
- In-season reports used by the COA to help determine sign-off eligibility at competitions without having to look up individual officials through the RTR

Requirements
============

The system will run on any Windows 10 or 11 system.  The system should also run on Linux but this has not been tested.
